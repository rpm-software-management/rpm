DbEnv::lock_id_free

 

#include <db_cxx.h>


int
DbEnv::lock_id_free(u_int32_t id);

Description

The DbEnv::lock_id_free method frees a locker ID allocated by the DbEnv::lock_id method.

The DbEnv::lock_id_free method either returns a non-zero error value or throws an exception that encapsulates a non-zero error value on failure, and returns 0 on success.

Errors

EINVAL
An invalid flag value or parameter was specified.

The locker ID is invalid or locks are still held by this locker ID.

The DbEnv::lock_id_free method may fail and throw an exception or return a non-zero error for errors specified for other Berkeley DB and C library or system methods. If a catastrophic error has occurred, the DbEnv::lock_id_free method may fail and either return DB_RUNRECOVERY or throw a DbRunRecoveryException, in which case all subsequent Berkeley DB calls will fail in the same way.

Class

DbEnv, DbLock
