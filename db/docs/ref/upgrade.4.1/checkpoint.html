Upgrading Berkeley DB Applications

Release 4.1: DB_CHECKPOINT, DB_CURLSN

The DB_CHECKPOINT flag has been removed from the DB_LOGC->get and DB_ENV->log_put functions. It is very unlikely application programs used this flag. If your application used this flag, please contact Sleepycat Software support for help in upgrading.

The DB_CURLSN flag has been removed from the DB_ENV->log_put function. It is very unlikely application programs used this flag. If your application used this flag, please contact Sleepycat Software support for help in upgrading.
