Upgrading Berkeley DB Applications

Release 4.1: DB_ENV->log_register

The DB_ENV->log_register and DB_ENV->log_unregister interfaces were removed from the Berkeley DB 4.1 release. It is very unlikely application programs used these interfaces. If your application used these interfaces, please contact Sleepycat Software support for help in upgrading.
