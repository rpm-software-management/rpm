Upgrading Berkeley DB Applications

Release 4.0: DB_ENV->lock_id_free

A new locker ID related API, the DB_ENV->lock_id_free function, was added to Berkeley DB 4.0 release. Applications using the DB_ENV->lock_id function to allocate locker IDs may want to update their applications to free the locker ID when it is no longer needed.
