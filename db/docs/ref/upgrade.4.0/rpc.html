Release 4.0: DB_ENV->set_server

The DB_ENV->set_server interface has been replaced with the DB_ENV->set_rpc_server function. The DB_ENV->set_server interface can be easily converted to the DB_ENV->set_rpc_server function by changing the name, and specifying a NULL for the added argument, second in the argument list.
