Release 3.2: db_dump

In previous releases of Berkeley DB, the db_dump utility dumped Recno access method database keys as numeric strings. For consistency, the db_dump utility has been changed in the 3.2 release to dump record numbers as hex pairs when the data items are being dumped as hex pairs. (See the -k and -p options to the db_dump utility for more information.) Any applications or scripts post-processing the db_dump output of Recno databases under these conditions may require modification.
